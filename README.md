## ðŸ” Overview
This Power BI report provides a clear analysis of **customer quotations** within the company.  
The goal is to understand which quotations are **Completed** and which are **Uncompleted**, and to support better follow-up and decision-making.

The report includes:

- **Main overview page** summarizing quotation distribution.
- A page for **Completed Quotations** only.
- A page for **Uncompleted Quotations** only.
- Each page is available in **Light Mode** and **Dark Mode**.
- **Navigation buttons** were added to allow smooth movement across report pages without confusion.

This structure ensures clarity, usability, and faster data interpretation for any team member.
